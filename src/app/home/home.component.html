Welcome to the Quote Generator App

This app has been created as a client to use the 150000+ Quotes API developed by chikaslocalhost from rapiapi.com

The API contains over 150000 quotes that you can obtain with our different methods

The app has three ways of giving you back a quote:

  • Random
    You'll get a random quote from the +150000 database
  • By author
    You can enter your favorite author's name and a quote from him/her will be returned randomly
  • By keywords
    You can enter any list of keywords to find a themed quote (ie. love, death, motivation)
